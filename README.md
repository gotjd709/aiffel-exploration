AIFFEL_Daejeon
===

2020년 12월 28일부터 2021년 06월 22일까지 AIFFEL_Daejeon 교육을 들으며 수행한 미니프로젝트들의 기록입니다.

### 소개
AIFFEL에서는 경쟁보다는 상생을 추구하는 자기주도적인 지원자들이 모여 Learning by Doing의 방식으로 배워나갑니다.

자세한 사항은 [여기](https://dj.aiffel.io/)를 클릭해주세요
